## 🎯 מה זה?

אפליקציה שעוזרת למנהלים להבין את **מצב העומס הכולל** של כל עובד בצוות — לא רק אזעקות, אלא התמונה המלאה.

## 📊 מדדים

| מדד | משקל | הסבר |
|-----|-------|------|
| 🚨 התרעות צבע אדום | לפי אזור מגורים | נתונים אמיתיים מ-API פיקוד העורף |
| 👶 ילדים וגילאים | תינוק ×2.0 → נער ×0.8 | ילדים קטנים = יותר לחץ במקלט |
| 🪖 בן/בת זוג במילואים | +20 נקודות | עומס כפול על ההורה שנשאר |
| 🚗 זמן נסיעה | +2.5 עד +10 | נסיעה ארוכה = שחיקה יומית |
| 🚧 פקקים מהמצב | +8 נקודות | חסימות כבישים, מחסומים |
| 🏚️ מפונה | +15 נקודות | חיים בחוסר ודאות |
| 💔 קרבה לאירוע | +18 נקודות | מכיר נפגעים, חטוף במשפחה |

## ✨ פיצ'רים

- חיבור API פיקוד העורף בזמן אמת
- מפת חום אזורית של התרעות
